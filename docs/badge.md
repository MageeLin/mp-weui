# Badge

> 徽章

## 引入

    import mpBadge from 'mp-weui/packages/badge'

    export default {
      components: {
        mpBadge
      }
    }

## 示例

指定颜色

    <mp-badge color="#888" />

指定文本

    <mp-badge text="20" />

## Props

名称 | 说明 | 类型 | 可选值 | 默认值
-- | -- | -- | -- | --
color | 显示颜色 | string |  | #e64340
text | 显示文本 | string  |  |
wrapClassName | 外层容器的类名 | string  |  |
